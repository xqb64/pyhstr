# pyhstr

Work in progress.

### Installation

```bash
git clone https://github.com/xvm32/pyhstr.git
cd pyhstr
python3 -m pip install -e .
```

### Usage

You should make an alias:

```bash
alias py='python3 -ic "from pyhstr.init import hh"'
```

Then:

```
>>> hh
```
